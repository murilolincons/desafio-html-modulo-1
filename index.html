O que são Clients?

[Voltar]

Clients (ou clientes) são os dispositivos ou softwares que pedem informações na web. No seu dia a dia, isso inclui: Navegador (Chrome, Firefox, Edge…), aplicativos de celular, programas que fazem requisições a APIs. Eles enviam pedidos (requisições) e recebem respostas — tudo em cima do protocolo HTTP/HTTPS.

O que são Servers?

[Voltar]

Server (ou servidor) é o computador que processa e devolve o que o client pediu. Recebe requisições, processa regras de negócio, consulta banco de dados e retorna respostas (páginas HTML, JSON, arquivos…).

Linguagem de Programação

[Voltar]

HTML é uma linguagem de programação uma linguagem de marcação, como o próprio nome já diz. Já a linguagem de programação é o idioma que usamos para dar comandos claros ao computador. Elas definem como escrevemos lógica, estruturas e comportamentos. Algumas das mais usadas atualmente:

  1. TypeScript
  2. Python
  3. JavaScript
  4. Java
  5. C#
Importante: Independente da linguagem, o ponto é: programar é dizer ao computador o que fazer e como fazer.
